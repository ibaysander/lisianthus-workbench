Tutorial Instalasi WSO2 API Manager di Windows 11

WSO2 API Manager adalah platform open-source yang digunakan untuk mengelola, mengatur, dan memantau penggunaan API. Dalam tutorial ini, kita akan membahas langkah-langkah instalasi WSO2 API Manager versi 4.4.0 di Windows 11 menggunakan JDK 21. Pastikan sistem Anda memenuhi prasyarat instalasi. Prasyarat Instalasi Sistem Operasi: Windows 11. Java Development Kit (JDK): JDK 11, 17, atau 21 (unduh dari situs resmi Oracle atau OpenJDK. RAM: Minimal 4 GB (2 GB untuk JVM dan 2 GB untuk Sistem Operasi). Ruang Disk: Minimal 10 GB. Port yang Tersedia: Pastikan port 9443, 8280, dan 8243 tidak digunakan oleh aplikasi lain. Langkah 1: Unduh WSO2 API Manager Kunjungi halaman GitHub WSO2 API Manager dan lihat bagian release. Unduh file ZIP wso2am-4.4.0.zip. Ekstrak file ZIP ke direktori pilihan Anda, misalnya D:\wso2am-4.4.0. Langkah 2: Instal JDK Unduh JDK. Misalkan pada kali ini kita akan menggunakan JDK 21. Unduh dari situs resmi Oracle atau OpenJDK. Jalankan installer JDK dan ikuti petunjuk instalasi. Setelah instalasi selesai, atur path JAVA_HOME: Buka System Properties > Advanced > Environment Variables. Klik New di bawah System Variables. Masukkan JAVA_HOME sebagai Variable Name dan path instalasi JDK (misalnya, C:\Program Files\Java\jdk-21) sebagai Variable Value. Tambahkan %JAVA_HOME%\bin ke variabel PATH. Verifikasi instalasi JDK dengan membuka Command Prompt dan menjalankan perintah: java --version Pastikan output menunjukkan versi JDK 21. Langkah 3: Konfigurasi WSO2 API Manager Buka direktori tempat Anda mengekstrak WSO2 API Manager, misalnya D:\wso2am-4.4.0. Edit file deployment.toml yang terletak di repository/conf/ untuk menyesuaikan konfigurasi: Port offset jika terdapat port yang telah digunakan oleh aplikasi lain [server] offset = 1 Akun super admin jika tidak ingin default admin-admin [super_admin] username = "admin" password = "admin" Langkah 4: Menjalankan WSO2 API Manager Buka Command Prompt dan arahkan ke direktori bin di dalam folder WSO2 API Manager: cd D:\wso2am-4.4.0\bin Jalankan perintah berikut untuk memulai WSO2 API Manager: api-manager.bat Tunggu hingga proses startup selesai. Anda akan melihat log yang menunjukkan server telah berjalan. Langkah 5: Mengakses Konsol WSO2 API Manager Buka browser dan akses konsol berikut: Publisher: https://localhost:9443/publisher Developer Portal: https://localhost:9443/devportal Admin Console: https://localhost:9443/admin Carbon Console: https://localhost:9443/carbon Gunakan kredensial default: Username: admin Password: admin Langkah 6: Mengubah Password Default Admin dari Carbon Masuk ke Carbon Console (https://localhost:9443/carbon). Navigasi ke Main > Identity > Users and Roles > List > Users. Pilih user admin dan klik Change Password. Masukkan password baru dan simpan perubahan.
