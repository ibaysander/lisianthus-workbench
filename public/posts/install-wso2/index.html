Tutorial Instalasi WSO2 API Manager di Windows 11 Menggunakan JDK 21#

WSO2 API Manager adalah platform open-source yang digunakan untuk mengelola, mengatur, dan memantau penggunaan API. Dalam tutorial ini, kita akan membahas langkah-langkah instalasi WSO2 API Manager versi 4.4.0 di Windows 11 menggunakan JDK 21.

Pastikan Anda telah mempersiapkan sistem Anda dengan baik sebelum memulai.

Prasyarat Instalasi#

  1. Sistem Operasi: Windows 11.
  2. Java Development Kit (JDK): JDK 21 (unduh dari situs resmi Oracle).
  3. RAM: Minimal 4 GB (disarankan 8 GB atau lebih).
  4. Ruang Disk: Minimal 4 GB.
  5. Port yang Tersedia: Pastikan port 9443, 8280, dan 8243 tidak digunakan oleh aplikasi lain.

Langkah 1: Unduh WSO2 API Manager#

  1. Kunjungi halaman GitHub WSO2 API Manager.
  2. Unduh file ZIP wso2am-4.4.0.zip.
  3. Ekstrak file ZIP ke direktori pilihan Anda, misalnya D:\wso2am-4.4.0.

Langkah 2: Instal JDK 21#

  1. Unduh JDK 21 dari situs resmi Oracle.
  2. Jalankan installer JDK dan ikuti petunjuk instalasi.
  3. Setelah instalasi selesai, atur variabel lingkungan JAVA_HOME:
    • Buka System Properties > Advanced > Environment Variables.
    • Klik New di bawah System Variables.
    • Masukkan JAVA_HOME sebagai Variable Name dan path instalasi JDK (misalnya, C:\Program Files\Java\jdk-21) sebagai Variable Value.
    • Tambahkan %JAVA_HOME%\bin ke variabel PATH.
  4. Verifikasi instalasi JDK dengan membuka Command Prompt dan menjalankan perintah:
    java --version
    Pastikan output menunjukkan versi JDK 21.

Langkah 3: Konfigurasi WSO2 API Manager#

  1. Buka direktori tempat Anda mengekstrak WSO2 API Manager, misalnya D:\wso2am-4.4.0.
  2. Edit file deployment.toml yang terletak di repository/conf/ untuk menyesuaikan konfigurasi:
    • Atur hostname sesuai dengan nama domain atau IP server Anda:
      [server]
hostname = "localhost"
    • Jika Anda menggunakan database eksternal (seperti MySQL), sesuaikan konfigurasi database di bagian [database] .

Langkah 4: Menjalankan WSO2 API Manager#

  1. Buka Command Prompt dan arahkan ke direktori bin di dalam folder WSO2 API Manager:
    cd D:\wso2am-4.4.0\bin
  2. Jalankan perintah berikut untuk memulai WSO2 API Manager:
    api-manager.bat --start
  3. Tunggu hingga proses startup selesai. Anda akan melihat log yang menunjukkan server telah berjalan.

Langkah 5: Mengakses Konsol WSO2 API Manager#

  1. Buka browser dan akses konsol berikut:
    • Publisher: https://localhost:9443/publisher
    • Developer Portal: https://localhost:9443/devportal
    • Admin Console: https://localhost:9443/admin
    • Carbon Console: https://localhost:9443/carbon
  2. Gunakan kredensial default:
    • Username: admin
    • Password: admin

Langkah 6: Mengubah Password Default#

  1. Masuk ke Carbon Console (https://localhost:9443/carbon).
  2. Navigasi ke Main > User Management > Users.
  3. Pilih user admin dan klik Change Password.
  4. Masukkan password baru dan simpan perubahan.

Langkah 7: Mengatur Domain Khusus#

Jika Anda ingin mengubah localhost menjadi nama domain khusus (misalnya, example.com), ikuti langkah berikut:

  1. Edit file deployment.toml di repository/conf/:
    [server]
hostname = "example.com"
  2. Restart WSO2 API Manager untuk menerapkan perubahan .

Langkah 8: Menghentikan WSO2 API Manager#

  1. Buka Command Prompt dan arahkan ke direktori bin.
  2. Jalankan perintah berikut untuk menghentikan server:
    api-manager.bat --stop

Kesimpulan#

Dengan mengikuti tutorial ini, Anda telah berhasil menginstal dan menjalankan WSO2 API Manager versi 4.4.0 di Windows 11 menggunakan JDK 21. Anda dapat mulai mengelola API Anda melalui konsol yang disediakan.

Selamat mencoba!
